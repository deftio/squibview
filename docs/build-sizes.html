Build Configuration Guide

All builds include autoload capability (v1.0.18+). Available in both ESM and UMD formats:

ConfigurationWhat It DoesWhen to UseLibraries Included
StandardIncludes autoload for CDN librariesDefault choice for most projectsCore editor with autoload capability
LeanMinimal build, you provide all dependenciesAdvanced users with existing bundler setupNone - bring your own
StandaloneEverything bundled, no external dependenciesOffline/airgapped environmentsAll libraries pre-bundled (~3.5MB)

Quick Decision Guide

  • Need it to just work? → Use Standard (258.8 KB with autoload)
  • Have a complex build system? → Use Lean (139.7 KB, you control dependencies)
  • No internet/CDN access? → Use Standalone (3576.3 KB, everything included)
