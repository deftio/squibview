Release Process

Current Process

‚úÖ Automated tooling: tools/make-release.sh script handles git tagging and GitHub releases ‚úÖ Version synchronization: Automatic sync between package.json and src/version.js ‚úÖ Pre-flight checks: Validates git status, authentication, and branch ‚úÖ Manual npm publish: Prevents accidental publishes with dry-run preview

Release Checklist

  1. Update package.json version
  2. Run npm run prerelease (runs tests and builds)
  3. Add release notes entry to this file
  4. Run ./tools/make-release.sh "release notes"
  5. Test npm package locally
  6. Publish to npm with npm publish
  7. Update documentation if needed
